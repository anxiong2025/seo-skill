## حالات الاستخدام

| السيناريو | مشكلتك | ما يفعله الـ Skill |
|:------|:---------|:---------------|
| 📝 **مدونة / مطور مستقل** | محتوى رائع لكن Google لا يفهرسه و ChatGPT لا يستشهد به | فحص SEO تقني شامل + تحسين GEO + مراجعة حداثة المحتوى |
| 💼 **SaaS / موقع منتج** | المنافسون يهيمنون على `/vs/xxx`، صفحات الميزات لا تُصنَّف | صفحات مقارنة المنافسين + التكاملات + Pricing SSR + أصول روابط |
| 🛒 **متجر إلكتروني** | صفحات منتجات مكررة، رفض في Merchant Center، غياب الزيارات الموسمية | Product Schema + إزالة تكرار المتغيرات + Shopping Feed + تقويم العروض |
| 📖 **توثيق المطورين** | API docs هي SPA — Googlebot لا يرى شيئاً، لا llms.txt | استراتيجية إصدارات + SEO مقتطفات الكود + llms.txt |
| 📍 **نشاط تجاري محلي** | لا يظهر في Google Maps، معلومات NAP متناثرة | LocalBusiness Schema + اتساق NAP + صفحات متعددة المواقع |
| 🌏 **عالمي / عابر للحدود** | دخول أسواق جديدة بدون تصنيفات، E-E-A-T ضعيف، لا ظهور في بحث الذكاء الاصطناعي | حزمة SEO عابرة للحدود + استراتيجية النطاق + hreflang + GEO متعدد الأسواق |
| 🎬 **محتوى / فيديو / إعلام** | مواقع الفيديو والأخبار والدورات تحتاج Schema متخصص وإشارات الحداثة | Video SEO + NewsArticle + Course Schema + استراتيجية التحديث |

**إذا كان موقعك يعمل بكود، يمكن لهذا الـ Skill مراجعته.**

## عرض توضيحي

```
المستخدم ❯ /seo

Claude   ❯ تم الكشف عن: مشروع Astro + موقع SaaS (/features/, /pricing/, /integrations/)
           تطبيق إعداد أوزان SaaS، بدء التدقيق...

           ## تقرير تدقيق SEO — yoursaas.com
           ## التقييم: 62/100 (D) — نوع الموقع: SaaS / منتج

           ### 🔴 حرج (3)
           1. صفحة Pricing تعتمد كلياً على JS — Googlebot يرى "Loading..."
              → التحويل إلى SSR أو generateStaticParams
           2. لا توجد صفحات /vs/منافس/ — المنافسون يهيمنون على جميع نتائج المقارنة
              → إنشاء أفضل 5 صفحات مقارنة مع المنافسين
           3. llms.txt مفقود — زاحفات الذكاء الاصطناعي لا تفهم بنية المنتج
              → إنشاء public/llms.txt

           ### 🟠 عالي (5)
           1. لا يوجد دليل /integrations/ — خسارة زيارات "منتج + أداة" ذات الذيل الطويل
           2. لا توجد أدوات مجانية كأصول روابط — أقوى قناة روابط لـ SaaS غير مستخدمة
           3. 12 صفحة ميزات تشترك في نفس og:image — إنشاء OG فريد لكل صفحة
           4. lastmod في Changelog هو 2025-08 — إشارة الحداثة تتراجع
           5. عناوين 5 صفحات ميزات لا تحتوي على الكلمات المفتاحية المستهدفة

           ما المشكلات التي تريد إصلاحها؟ اكتب "إصلاح الكل" أو اختر فئة.
```

## التثبيت

### الخيار الأول: npx skills (متوافق مع Cursor و Cline وغيرها)

```bash
npx skills add anxiong2025/seo-skill
```

> الملفات في `.agents/skills/seo/`. إذا لم يُعرَّف `/seo`، استخدم الخيار الثاني.

### الخيار الثاني: تثبيت يدوي (موصى به لـ Claude Code)

```bash
mkdir -p .claude/skills/seo
curl -sL https://raw.githubusercontent.com/anxiong2025/seo-skill/main/SKILL.md \
  -o .claude/skills/seo/SKILL.md
```

```bash
# اختياري: تحميل مستندات مرجعية
mkdir -p .claude/skills/seo/references
for f in keyword-research-guide competitor-analysis-guide schema-templates meta-tag-spec og-image-guide; do
  curl -sL "https://raw.githubusercontent.com/anxiong2025/seo-skill/main/references/${f}.md" \
    -o ".claude/skills/seo/references/${f}.md"
done
```

| الطريقة | مثال | الوصف |

| **تدقيق كامل** | `/seo` | تشغيل جميع الوحدات المناسبة، إخراج النتيجة + التقرير الكامل |

| **لغة طبيعية** | `ابحث عن كلمات مفتاحية`، `حلل منافسي صفحة التسعير`، `حسّن اقتباسات الذكاء الاصطناعي` | صف ما تحتاجه، Claude يختار الوحدة المناسبة |

### ثلاثة أنماط تدقيق

| النمط | المتطلبات | التغطية | الأنسب لـ |
|:------|:----------|:------:|:------|
| **بدون اتصال** | لا شيء | ~60-70% | المحاولة الأولى، بدون تسجيل دخول |
| **قياسي** | Google Search Console + GA4 (كلاهما مجاني) | ~90% | **موصى به** |
| **كامل** | + Bing WMT + Keyword Planner + MCP | ~100% | فرق SEO الاحترافية |

> ابدأ بـ `/seo` للحصول على تدقيق كامل وتقييم. ثم دقق بالوحدات حسب الحاجة.

## ما يتم فحصه

**37 وحدة · 700+ فحص · تقييم 0-100 + درجة A-F · ترجيح ديناميكي حسب نوع الموقع**

<details open>
<summary><strong>📋 قائمة الوحدات الكاملة</strong></summary>

<br>

#### الأساس والتقني (جميع المواقع)

| الوحدة | الفحوصات الرئيسية |
|:------|:--------|
| **الأساس** | GA4، التحقق من GSC، noindex، إعداد sitemap |
| **SEO التقني** | robots.txt، canonical، HTTPS، إعادة التوجيه، الجوال |
| **On-Page SEO** | title، description، تسلسل H1، alt للصور، روابط داخلية، بنية URL |
| **البيانات المنظمة** | 12 نوع JSON-LD (Article / Product / FAQ / HowTo / LocalBusiness، إلخ) |
| **الأداء / CWV** | LCP، INP، CLS، الخطوط، ضغط الصور، بيانات CrUX الحقيقية |
| **JS Rendering / SSR** | التصيير من جانب الخادم، hydration، fallback SPA |
| **التحقق من HTML** | DOCTYPE، charset، بنية head، علامات مكررة |
| **أمان SEO** | HTTPS/HSTS، CSP، مفاتيح مكشوفة، محتوى مختلط |
| **إمكانية الوصول** | ARIA، تباين الألوان، تسميات النماذج، التنقل بالكيبورد |
| **Favicon** | favicon.ico / .svg، apple-touch-icon |

#### عصر الذكاء الاصطناعي (GEO + المحتوى)

| الوحدة | الفحوصات الرئيسية |
|:------|:--------|
| **GEO (اقتباس الذكاء الاصطناعي)** | llms.txt، وصول زاحفات الذكاء الاصطناعي، تحسين الكيانات، الترميز الدلالي |
| **SEO المحتوى** | إشارات E-E-A-T، بنية المحتوى، FAQ، جودة صفحات الهبوط |
| **السلطة الموضوعية** | مجموعات Hub-Spoke، فجوات المحتوى، اكتمال الموضوع |
| **حداثة المحتوى** | تحديثات dateModified، اكتشاف التراجع، استراتيجية التجديد |
| **بحث الكلمات المفتاحية** | توسيع البذور، الذيل الطويل، نية البحث، الأكل الذاتي |

#### العمليات (مبني على البيانات)

| الوحدة | الفحوصات الرئيسية |
|:------|:--------|
| **تكامل GSC** | تحليل الاستعلامات/CTR/الموضع، استخراج الفرص |
| **استرداد العقوبات** | Manual Action، تراجع HCU، تشخيص Core Update |
| **ميزانية الزحف** | تحليل السجلات للمواقع الكبيرة (>1000 صفحة) |
| **هجرة الموقع** | تعيين 301، سلاسل إعادة التوجيه، مقارنة ما قبل/بعد |
| **تشخيص التصنيف** | لماذا لا تُصنَّف الصفحات — تحليل عميق للكلمة المفتاحية → URL |
| **تحليل المنافسين** | ميزات SERP، تمييز المحتوى، فرص المقتطفات المميزة |
| **بناء الروابط** | بنية الروابط الداخلية، نص رابط، بحث التواصل |
| **استرداد الروابط** | إشارات العلامة التجارية بدون رابط، روابط خلفية معطلة |
| **OG / وسائل التواصل** | og:image (1200×630)، Twitter Card |
| **اختبار A/B** | منهجية تجارب العنوان / meta / البنية |
| **CRO لـ SEO** | تحويل صفحات الهبوط، CTA، إشارات الثقة |

#### خاص بالصناعة (يُفعَّل تلقائياً حسب نوع الموقع)

| الوحدة | يُفعَّل عند |
|:------|:---------|
| 🛒 **SEO التجارة الإلكترونية** (40+ فحص) | وجود صفحات منتجات / سلة تسوق |
| 💼 **SaaS / موقع منتج** | وجود /features /pricing /integrations |
| 📖 **توثيق المطورين** | وجود /docs /api، محتوى مُصدَّر بإصدارات |
| 🌏 **عالمي / عابر للحدود** | أهداف متعددة اللغات / المناطق |
| 📍 **SEO المحلي** | موقع LocalBusiness |
| 🌐 **دولي متعمق** | موقع hreflang متعدد اللغات |
| 🎬 **SEO الفيديو** | موقع يركز على الفيديو |
| 👥 **UGC / برمجي** | موقع مجتمع / مجمِّع |

</details>

> **التقييم**: كل فحص هو Pass / Partial / Fail → مجموع مرجح → تقييم 0-100.
> **حد الفشل الحرج**: المشكلات الحرجة (sitemap مفقود، robots محجوب، ≥60% عناوين مكررة) تحد التقييم عند 60.
